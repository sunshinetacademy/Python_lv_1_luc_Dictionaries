⚠️ مشكلة Lists:

الوصول للعناصر يكون بالأرقام (index)، مما يصعب تذكر موقع كل عنصر!

📖 ما هو الـ Dictionary؟

الـ Dictionary هو بنية بيانات تخزن المعلومات في شكل أزواج من المفتاح والقيمة (Key-Value Pairs).

التشبيه:

مثل القاموس الحقيقي: كل كلمة (مفتاح) لها معنى (قيمة).

name
Ahmed
age
15
city
Cairo

مثال بسيط:

student = { "name": "Ahmed", "age": 15, "grade": "A" }
💡 المميزات:
  • الوصول بالاسم (مفتاح) بدلاً من الرقم
  • سهل القراءة والفهم
  • مرن وقابل للتعديل

الطريقة الأساسية:

🔑 الوصول للقيم

الطريقة الأولى: الأقواس المربعة

person = { "name": "Ahmed", "age": 15 } print(person["name"]) # Ahmed print(person["age"]) # 15

الطريقة الثانية: get() (أكثر أماناً)

print(person.get("name")) # Ahmed print(person.get("phone")) # None print(person.get("phone", "N/A")) # N/A

➕ الإضافة والتعديل

إضافة مفتاح جديد:

person = {"name": "Ahmed"} # إضافة مفتاح جديد person["age"] = 15 person["city"] = "Cairo" print(person)

تعديل قيمة موجودة:

person = { "name": "Ahmed", "age": 15 } # تعديل العمر person["age"] = 16 print(person["age"]) # 16

➖ حذف العناصر

الطريقة الأولى: del

person = { "name": "Ahmed", "age": 15, "city": "Cairo" } del person["city"] print(person)

الطريقة الثانية: pop()

person = { "name": "Ahmed", "age": 15 } # حذف وإرجاع القيمة age = person.pop("age") print("Removed age:", age) print(person)

🎓 اختبار الفهم

السؤال 1: كيف ننشئ Dictionary؟

أ) d = ["a": 1]
ب) d = {"a": 1}
ج) d = ("a": 1)

السؤال 2: كيف نصل لقيمة "name"؟

person = {"name": "Ali", "age": 20}
أ) person["name"]
ب) person[0]
ج) person.name

السؤال 3: ماذا ترجع dict.keys()؟

أ) كل المفاتيح
ب) كل القيم
ج) كل الأزواج

السؤال 4: كيف نتحقق من وجود مفتاح؟

أ) if "key" in dict:
ب) if dict.has("key"):
ج) if dict.exists("key"):

السؤال 5: ما ناتج الكود؟

d = {"a": 1, "b": 2} for k, v in d.items(): print(k, v)
أ) a b
ب) a 1 ثم b 2
ج) 1 2
