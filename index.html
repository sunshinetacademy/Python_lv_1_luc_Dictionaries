📖 ما هو الـ Dictionary؟

الـ Dictionary هو قاموس سحري بيخزن البيانات في شكل أزواج: مفتاح وقيمة!

🔑💎

💡 التشبيه الذهني:

📚 القاموس الحقيقي

كلمة → معناها

cat → قطة

dog → كلب

📱 دفتر العناوين

الاسم → الرقم

Ahmed → 01012345

Sara → 01198765

📝 أول Dictionary:

student = { "name": "Ahmed", "age": 15, "grade": 90 }
🔑 name
💎 Ahmed
🔑 age
💎 15
🔑 grade
💎 90
🌟 المصطلحات:
  • Key (المفتاح): الاسم اللي بتدور بيه (زي "name")
  • Value (القيمة): المعلومة نفسها (زي "Ahmed")
  • Key-Value Pair: الزوج الكامل (مفتاح + قيمة)

⚔️ Lists vs Dictionaries

إمتى نستخدم كل واحدة؟

المعيار List 🎁 Dict 📖
الاستخدام مجموعة من نفس النوع بيانات بوصف (key-value)
الوصول بالرقم (index) بالاسم (key)
الترتيب مهم مش مهم (من Python 3.7+)
مثال [1, 2, 3] {"a": 1, "b": 2}
البحث بطيء (O(n)) سريع جداً (O(1))
🌟 القاعدة الذهبية:
  • 📋 List: "عندي مجموعة أرقام/أسماء"
  • 📖 Dict: "عندي بيانات ليها وصف (اسم، عمر، عنوان)"
  • 🪆 الاتنين مع بعض: قائمة من الأشخاص!

🎓 اختبار الفهم الشامل

8 أسئلة لقياس فهمك الكامل!

❓ السؤال 1: كيف ننشئ Dictionary؟

أ) d = ["a": 1]
ب) d = {"a": 1}
ج) d = ("a": 1)
د) d = <"a": 1>

❓ السؤال 2: كيف نصل لقيمة "name"؟

person = {"name": "Ali", "age": 20}
أ) person["name"]
ب) person[0]
ج) person.name
د) person(name)

❓ السؤال 3: كيف نضيف مفتاح جديد؟

أ) d["new"] = value
ب) d.add("new", value)
ج) d.insert("new", value)
د) d.append("new", value)

❓ السؤال 4: ماذا ترجع dict.keys()؟

أ) كل المفاتيح
ب) كل القيم
ج) كل الأزواج
د) عدد العناصر

🎓 اختبار الفهم (تابع)

❓ السؤال 5: كيف نتحقق من وجود مفتاح؟

أ) if "key" in dict:
ب) if dict.has("key"):
ج) if dict.contains("key"):
د) if dict.exists("key"):

❓ السؤال 6: ما ناتج الكود؟

d = {"a": 1, "b": 2} for k, v in d.items(): print(k, v)
أ) a b
ب) a 1
b 2
ج) 1 2
د) Error

❓ السؤال 7: ما الفرق بين [] و .get()؟

أ) .get() يرجع None لو المفتاح مش موجود
ب) [] أسرع
ج) مفيش فرق
د) .get() للقيم بس

❓ السؤال 8: إمتى نستخدم Dict؟

أ) لما يكون عندنا أرقام كتير
ب) لما نحتاج نخزن بيانات بأسماء
ج) دايمًا
د) أبدًا
