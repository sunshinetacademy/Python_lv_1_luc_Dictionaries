المستوى 2 🟡 الوصول للقيم

🐍 مثال 4: الوصول بمفتاح

student = { "name": "Sara", "age": 14, "grade": "A" } print(student["name"]) # Sara print(student["age"]) # 14 print(student["grade"]) # A

🐍 مثال 5: استخدام get() - الأأمن!

student = { "name": "Sara", "age": 14 } # الطريقة الأولى: [] (خطأ لو المفتاح مش موجود) print(student["name"]) # Sara ✅ # الطريقة الثانية: .get() (آمنة) print(student.get("phone")) # None (مش Error) print(student.get("phone", "غير موجود")) # غير موجود
🎮 جربها بنفسك
game = { "title": "Minecraft", "year": 2011, "rating": 9.5 } print("Game:", game["title"]) print("Year:", game["year"]) print("Rating:", game.get("rating")) print("Price:", game.get("price", "Free"))

المستوى 3 🔴 الإضافة والتعديل

🐍 مثال 6: إضافة مفتاح جديد

person = {"name": "Ali"} # إضافة مفتاح جديد person["age"] = 12 person["city"] = "Cairo" print(person) # {'name': 'Ali', 'age': 12, 'city': 'Cairo'}

🐍 مثال 7: تعديل قيمة موجودة

person = { "name": "Ali", "age": 12 } # تعديل العمر person["age"] = 13 print(person["age"]) # 13
🎮 جربها بنفسك
bag = {} bag["book"] = "Math" bag["pen"] = "Blue" bag["eraser"] = "White" print("Before:", bag) # غير القلم bag["pen"] = "Red" print("After:", bag)

المستوى 4 🗑️ حذف العناصر

🐍 مثال 8: استخدام del

person = { "name": "Ali", "age": 12, "city": "Cairo" } del person["city"] print(person) # {'name': 'Ali', 'age': 12}

🐍 مثال 9: استخدام pop()

person = { "name": "Ali", "age": 12 } # حذف وإرجاع القيمة age = person.pop("age") print("Removed:", age) # 12 print(person) # {'name': 'Ali'}
🎮 جربها بنفسك
fruits = { "apple": 5, "banana": 3, "orange": 4 } print("Before:", fruits) del fruits["banana"] print("After del:", fruits) removed = fruits.pop("orange") print("Removed:", removed) print("After pop:", fruits)

🛠️ الدوال المفيدة

الدالة ماذا تفعل مثال
len() عدد العناصر len(person)
.keys() كل المفاتيح person.keys()
.values() كل القيم person.values()
.items() كل الأزواج ⭐ person.items()

🐍 مثال 10: تجربة كل الدوال

student = { "name": "Omar", "age": 13, "grade": "A" } # العدد print("Count:", len(student)) # المفاتيح print("Keys:", student.keys()) # القيم print("Values:", student.values()) # الأزواج print("Items:", student.items())

🔄 التكرار على Dictionary

🐍 مثال 11: استخدام items() مع for loop

scores = { "Math": 90, "Science": 85, "English": 92 } for subject, score in scores.items(): print(f"{subject}: {score}")

💡 ربط بالمحاضرات السابقة:

لاحظ إننا استخدمنا for (محاضرة 2) + items() (الجديد)!

🐍 مثال 12: حساب المجموع والمتوسط

🎮 جربها بنفسك
scores = { "Math": 95, "Science": 88, "English": 92, "History": 85 } total = 0 for subject, score in scores.items(): print(f"{subject}: {score}") total = total + score print("--------") print("Total:", total) print("Average:", total / len(scores))
